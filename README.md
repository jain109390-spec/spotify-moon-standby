This is a Spotify display with a moon phase fallback designed around the cheap yellow display
I made this because it was an interesting, yet practicaly starter model I wanted to test out. I am also weak in software and hardware, so having a guide would be very useful.

These are all the components; it will have a 3d printed casing that offers an ideal viewing angle by tilting it back at 9.29 degrees. 
All controls should be touch
Changing between moon phase and spotify display will be automatic
Spotify display will have volume control, pause, play, skip, back and progress bar
vents in the back keep the ESP chip cool, especially when downloading album art for spotify display.
I also added another hole for a speaker to possibly connect to in the future
The coding is mainly done by Claude, double-checked by me and Gemini
I got the original files for the case from https://makerworld.com/en/models/1259348-cheap-yellow-display-snap-case?from=search#profileId-1283404
